# 中文AI腔专项排查

以下为中文AI写作中最高频的腔调问题。读文本时逐条扫描，发现违禁项就地改掉，用更具体、更生活化的表达替换。

---

## 🔴 句式禁区

**禁止公式：**

| 违禁句式 | 典型特征 | 改写方向 |
|----------|----------|----------|
| 「不是 X，而是 Y」 | 否定式排比，强行拔高 | 直接说Y，或拆成两句陈述 |
| 「不仅仅是 X，更是 Y」 | 递进式拔高，AI最爱 | 去掉模板，直接说事实 |
| 「当……时」宏大起手 | 开篇就是"当AI时代来临""当数字化转型加速" | 直接进入具体场景 |
| 排比否定 | 连续三个"不是……不是……不是……" | 删掉，或用一句话概括 |

**改写示例：**
- ❌ 「这不仅仅是效率的提升，更是工作方式的革命」
- ✅ 「以前要三天，现在三十分钟。区别在这。」
- ❌ 「当数字化浪潮席卷各行各业，传统企业面临着前所未有的挑战」
- ✅ 「做了二十年外贸的张总，去年第一次用上了AI客服。」

---

## 🔴 判断禁区

**禁止用词：** 这很关键、这值得深思、归根结底、说到底、从本质上讲、本质上是

**问题：** 这些词是AI给读者"划重点"用的，真人不需要别人帮他判断什么值得深思。

**处理方式：** 直接删除。如果删除后前后不连贯，改写为自然的过渡句。

- ❌ 「这很关键——数据质量决定了模型上限。」
- ✅ 「数据质量决定了模型上限。」
- ❌ 「归根结底，用户要的是好用。」
- ✅ 「用户要的就是好用。」

---

## 🔴 语气禁区

| 违禁表达 | 处理方式 |
|----------|----------|
| 「我们」起笔（"我们正处在一个……"） | 删除"我们"，或改为具体主语 |
| 时代感拔高（"在XX时代背景下""随着XX的快速发展"） | 删除，直接进入正文 |
| 哲学升华收尾（"这或许正是……的意义所在"） | 删掉，用具体的事实或行动结尾 |
| 「无疑」「毋庸置疑」「不可否认」 | 删除，事实不需要强调 |

**改写示例：**
- ❌ 「我们正处在一个技术飞速迭代的时代。」
- ✅ （删掉，直接从下一句开始）
- ❌ 「这或许正是数字化转型的真正意义所在。」
- ✅ 「公司今年在数字化上投了270万，省了400万。」

## 🔴 词汇禁区

以下词汇只要出现就改掉，不存在"看情况保留"：

| 违禁词 | 替换方向 |
|--------|----------|
| 赋能 | 帮、让……可以、给……用上 |
| 闭环 | 流程走通、从头到尾搞定 |
| 赛道 | 领域、行业、方向 |
| 底层逻辑 | 根本原因、核心问题 |
| 重新定义 | 改变、没做到过的事 |
| 彻底颠覆 | 大幅改变、跟以前不一样 |
| 打法 | 方法、策略 |
| 抓手 | 切入点、突破口 |
| 对齐 | 统一、确认、沟通一致 |
| 颗粒度 | 细节、细致程度 |
| 痛点 | 问题、麻烦 |
| 落地 | 执行、做出来、上线 |
| 倒逼 | 迫使、催生 |
| 窗口期 | 机会、时机 |
| 全链路 | 从头到尾、全流程 |

**注意：** 替换时不按字面硬译，用具体语境下的自然汉语。

- ❌ 「通过AI赋能全链路，重塑行业底层逻辑」
- ✅ 「用AI把整个流程跑了一遍，发现中间三个环节可以省掉。」

## 🔴 修辞禁区

| 违禁修辞 | 典型表现 | 处理方式 |
|----------|----------|----------|
| 模板化比喻 | "犹如一匹黑马""像雨后春笋" | 改用具体事实 |
| 过度类比 | "就像当年互联网改变零售业一样" | 删掉类比，直接说事 |
| 空心对仗金句 | 「X 越…，Y 越…」格式 | 拆开说，或直接删 |
| 万能金句 | "在不确定性中寻找确定性""长期主义" | 换成说了什么具体做了什么 |

**改写示例：**
- ❌ 「技术越成熟，应用越广泛，挑战也越严峻。」
- ✅ 「技术现在成熟了，用的人多了，但问题也跟着来了：去年出了三次数据泄露。」
- ❌ 「在不确定性中寻找确定性。」
- ✅ 「老板的原话是：不管怎样，先把现金流保住。」
